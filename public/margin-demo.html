Component 1

Component 2

3. Large Margin (margin: "lg" = my-12)

Component 1

py-16 px-4 my-12

Component 2

4. Extra Large Margin (margin: "xl" = my-16)

Component 1

Legend:

Blue dashed outline = Section (includes padding + margin)
Green solid border = Content area (inside padding)

Tailwind Classes Used:

  • my-4 = margin-top: 1rem; margin-bottom: 1rem;
  • my-8 = margin-top: 2rem; margin-bottom: 2rem;
  • my-12 = margin-top: 3rem; margin-bottom: 3rem;
  • my-16 = margin-top: 4rem; margin-bottom: 4rem;
